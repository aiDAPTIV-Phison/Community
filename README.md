Official community hub for **Phison AI Data Platform** — technical Q&A, product discussions, feature ideas, and best-practice sharing.

This repository contains **no application source code**. Its primary purpose is [**GitHub Discussions**](https://github.com/AIDataPlatform/Community/discussions). Please post in Discussions rather than opening Issues (Issues are reserved for reproducible defects).

## How to Participate

1. You need a GitHub account ([sign up free](https://github.com/signup)).
2. Go to [**Discussions**](https://github.com/AIDataPlatform/Community/discussions).
3. Pick the right **Category** and write a clear title and body.
4. For technical questions, include: product model, software/firmware version, environment, and steps already tried.
5. For feature ideas, describe the use case and expected benefit.

**Search first** — check for existing threads before posting.

## Suggested Categories

Admins can enable and create these in **Settings → General → Discussions**:

| Category | Purpose | Example topics |
|----------|---------|----------------|
| **📣 Announcements** | Official news, events, community policy | Launch, maintenance windows |
| **📋 Release Notes** | Version updates and changelogs | Firmware, software, docs |
| **🗄️ Cache Server** | aiDAPTIV+ Cache Server (GPU KV Cache) | Capacity planning, GPU connectivity, hit rate |
| **💾 Storage Server** | PASCARI Storage Server | Block / File / Object access, cluster config |
| **🖥️ HCI** | Hyper-Converged Infrastructure | Resource orchestration, GPU grid, HCI deployment |
| **📐 Platform & Architecture** | Platform design, sizing, integration | Platform Sizing, multi-product topology |
| **❓ Q&A** | Cross-product technical questions | Install, troubleshooting, compatibility |
| **💡 Ideas** | Feature requests and voting | New protocols, APIs, management UI |
| **📚 Guides** | Tutorials, best practices, templates | Initial deployment, performance tuning |
| **🏆 Showcase** | Case studies and architecture stories | Customer POCs, benchmarks |

Add **Labels** when available (e.g. `cache-server`, `storage-server`, `hci`).

## Community Guidelines

- **Respectful and professional** — be kind to people, rigorous about technology.
- **Public discussion first** — do not post confidential or customer-sensitive information here unless a separate NDA channel exists.
- **Constructive feedback** — include context for suggestions; pair criticism with alternatives.
- **No duplicate posts** — reply on existing threads when the topic already exists.
- **Mark resolved** — when a question is solved, say so for future searchers.

Content that violates these guidelines may be hidden or removed; repeat offenders may be restricted.

---

## Issue vs Discussion

| Situation | Recommended channel |
|-----------|----------------------|
| Usage questions, architecture talk, guides | **Discussions** |
| Feature ideas, polls, roadmap input | **Discussions → Ideas** |
| Website typos, broken links (reproducible) | [AI-Data-Plateform Issues](https://github.com/AIDataPlatform/AI-Data-Plateform/issues) |
| Security vulnerabilities | Phison official security reporting (**do not** publish PoCs publicly) |
