# AI Data Platform Community

**Phison AI Data Platform** 的官方社群入口 — 技术交流、产品讨论、功能建议与最佳实践分享。

本 repository **不含应用程序源代码**；主要用途为 [**GitHub Discussions**](https://github.com/AIDataPlatform/Community/discussions)。请在 Discussions 发文，而非开 Issue（Issue 保留给可重现的缺陷报告）。

## 讨论区怎么用

1. 需要 GitHub 账号（[免费注册](https://github.com/signup)）。
2. 前往 [**Discussions**](https://github.com/AIDataPlatform/Community/discussions)。
3. 选择合适的 **Category**，撰写清楚标题与内容。
4. 技术问题请附上：产品型号、软件／固件版本、环境描述、已尝试步骤。
5. 功能建议请说明使用场景与预期效益。

**搜索优先**：发文前先搜索是否已有类似讨论。

## 社群守则

- **尊重与专业**：对人友善，对技术严谨。
- **公开讨论优先**：除非另有 NDA 渠道，请勿在此张贴机密或客户敏感信息。
- **建设性反馈**：功能建议请附场景；批评请附可行替代方案。
- **不重复发文**：相似主题请在既有讨论串回复或引用。
- **标记已解决**：问题解决后请回复说明，方便他人搜索。
